# ✂️ ClipMind — YouTube Clip Summarizer

> Drag to select any part of a YouTube video and get instant AI summary with bullet points, multiple languages, and PDF export.

## ✨ Features
- Drag to select any clip on the YouTube progress bar
- AI-powered summary using Groq (free, fast)
- Multiple languages — English, Bengali, Arabic, Hindi, Urdu, Spanish, French, Turkish
- Ask questions about the clip (Chat tab)
- Notebook — save and manage your summaries
- PDF export and copy/share support
- Daily usage tracker — 10 free summaries/day

## 🚀 Installation
1. Download or clone this repo
2. Go to chrome://extensions/
3. Enable Developer Mode
4. Click Load unpacked → select this folder
5. Get a free Groq API key at console.groq.com/keys
6. Open any YouTube video and start summarizing!
